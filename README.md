# NutriOne

Uma aplicação inteligente para cálculo de Taxa Metabólica Basal (TMB) e planejamento nutricional personalizado.

## 📚 Conceito Teórico

### O que é Taxa Metabólica Basal (TMB)?

A **Taxa Metabólica Basal** é a quantidade mínima de calorias que seu corpo consome em repouso absoluto para manter as funções vitais essenciais, como:

- Respiração
- Circulação sanguínea
- Funcionamento do sistema nervoso
- Síntese proteica
- Manutenção celular

A TMB **não inclui** atividades físicas, digestão ou termogênese induzida pela alimentação. É, portanto, o gasto energético basal do organismo em condições de descanso total.

---

## 🎯 Propósito do NutriOne

O NutriOne permite calcular sua **TMB personalizada** com base em dados biométricos científicos:

### Dados de Entrada
- **Peso** (em kg)
- **Altura** (em cm)
- **Idade** (em anos)
- **Sexo** (masculino/feminino)

### Metodologia
Utilizamos a **Fórmula de Mifflin-St Jeor**, considerada o padrão ouro para estimar TMB em adultos saudáveis. Esta fórmula é mais precisa que métodos anteriores como Harris-Benedict, especialmente para populações modernas.

**Fórmula:**
- **Homens:** TMB = (10 × peso) + (6,25 × altura) - (5 × idade) + 5
- **Mulheres:** TMB = (10 × peso) + (6,25 × altura) - (5 × idade) - 161

## 💡 Aplicações Práticas

Conhecer sua TMB é fundamental para:

1. **Definição de Metas Calóricas**
   - Calcular superávit calórico para ganho de massa
   - Definir déficit calórico seguro para perda de peso
   - Manter peso através de ingestão adequada

2. **Planejamento Nutricional**
   - Personalizar dietas com base no gasto energético real
   - Evitar restrições calóricas excessivas ou insuficientes
   - Otimizar distribuição de macronutrientes

3. **Compreensão do Metabolismo**
   - Entender variações individuais no gasto energético
   - Identificar fatores que influenciam o metabolismo
   - Tomar decisões informadas sobre nutrição

4. **Base para Cálculos Nutricionais**
   - TMB × Fator de Atividade = Gasto Energético Total (GET)
   - GET ± 300-500 kcal = Meta calórica para ganho/perda
